# Lazuriteで受信したメッセージに従ってNeoPixelを制御するプログラム

## 注意事項
NeoPixelの長さ60連です。必要に応じてLED_LENGTHのパラメータを変更してください。
消費電力削減のために指定したカラーコードを減色して使用しています。

## 使い方
1. 920MHz無線で受信したコードに従ってNeoPixelを制御します。
テキスト形式で0xXXYYYYYYと送信するだけです。
### XXの部分
01:  即座に色を変えます

02:  現在の色から次の色まで徐々に色を変化させていきます

03:  基板側から後ろに指定した色をシフトしていきます。

04:  後ろから基板側に指定した色をシフトしていきます。

05:  基板側から後ろに指定した色を一つだけ飛ばします。

06:  後ろから基板側に指定した色を一つだけ飛ばします。

07: 薄い光の中を、濃い光が動きます。

0B: 全て同時に、薄く光る、濃く光るを繰り返します。


### YYYYYYの部分
カラーコードを指定します。先頭２文字が赤、真ん中の２文字が緑、最後の二文字が青を示します。

### おまけ
"-1"をテキストで送信すると、動く虹色になります。
